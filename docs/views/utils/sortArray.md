## 洗牌算法

>原理\
>1.倒序循环这个数组\
>2.取范围从 0 到 i 的随机索引 j\
>3. i 与 j 位置交换\
>4.直到循环至数组的首个元素
```js
function shuffle(arr) {
  let i = arr.length;
  for(let i = arr.length-1; i > 0;i--) {
    let j = Math.floor(Math.random() * i);
    var t = arr[i]
    arr[i] = arr[j]
    arr[j] = t
  }
  return arr;
}
var arr = [1,2,3,4,5,6,7,8,9];
console.log(shuffle(arr)); //  [6, 3, 9, 1, 8,7, 2, 4, 5]
```
>看一下变换过程 随机数向下取整

## es6进阶
```js
function shuffle(arr) {
  let i = arr.length;
  while (i) {
    let j = Math.floor(Math.random() * i--);
    [arr[j], arr[i]] = [arr[i], arr[j]];
  }
  return arr;
}
```
